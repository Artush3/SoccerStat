# ⚽ SoccerStat

Приложение для просмотра футбольной статистики — лиги, команды, матчи.

## 🛠 Стек технологий

- **React 19** + **TypeScript**
- **SCSS Modules** — стилизация с CSS-переменными
- **Zustand** — управление состоянием
- **React Router v7** — маршрутизация
- **FSD** (Feature-Sliced Design) — архитектура
- **ESLint** + **Stylelint** + **Prettier** — качество кода
- **Vite** — сборщик
- **Vercel** — хостинг и серверные функции (прокси)

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/Artush3/SoccerStat.git
cd SoccerStat
```

2. Установка зависимостей

```bash
npm install
```

3. Получить API-токен

Зарегистрируйтесь на football-data.org и получите бесплатный токен.

4. Настройка переменных окружения

Создайте файл .env.local в корне проекта и добавьте строку:

```
VITE_USE_PROXY=true
```

Важно: Сам API-токен не нужно хранить локально — он уже находится в облачном проекте Vercel и будет загружен автоматически на следующем шаге.

5. Подключение к Vercel (только при первом запуске)

Убедитесь, что у вас установлен Vercel CLI:

```bash
npm install -g vercel
```

Свяжите локальную папку с проектом на Vercel и загрузите секретный токен:

```bash
vercel link          # выберите команду и проект soccer-stat
vercel env pull .env.local   # загрузит FOOTBALL_DATA_API_KEY в .env.local
```

Теперь в .env.local появится переменная FOOTBALL_DATA_API_KEY.

6. Запуск локального сервера разработки

```bash
vercel dev
```

Приложение будет доступно по адресу http://localhost:3000.

Обратите внимание:

· Обычный npm run dev не будет работать для API-запросов, так как требуется серверная функция-прокси.
· При включённом VPN или прокси-расширениях браузера сайт может работать некорректно из-за блокировок football-data.org. Рекомендуется отключать VPN.

🌐 Деплой

Проект автоматически деплоится на Vercel при каждом пуше в ветку main. Для ручного деплоя используйте:

```bash
vercel --prod
```

⚠️ Известные проблемы

· VPN и прокси: football-data.org может блокировать запросы из некоторых регионов или при использовании VPN. Если сайт не загружает данные, попробуйте отключить VPN.
· CORS: Прямые запросы к API из браузера невозможны без прокси. Локальная разработка возможна только через vercel dev.
